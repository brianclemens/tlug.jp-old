1. Itinerary

1. Presentations

1. The CAcert project (by [Yamada](Taisuke)(/taisuke yamada/))

1. Embedded Linux development (by [Povey](Jon)(/jon povey/))

The first speaker [Yamada](Taisuke)(/taisuke yamada/) will talk about the CAcert project
which is a community-based effort to build RootCA (Certificate
Authority).  [Yamada](Taisuke)(/taisuke yamada/) is currently working with CAcert.org to
organize the is first official CAcert event in Japan.

The second speaker [Povey](Jon)(/jon povey/) will talk about Embedded Linux development.
The talk will include a general overview of the process from board
layout end product, with an emphasis on the firmware, in an attempt to
answer the "how the crap does that work anyway" questions.
