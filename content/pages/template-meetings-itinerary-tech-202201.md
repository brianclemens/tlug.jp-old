1. Presentations:

*Headscale mesh network orchistration server* - Edward Middleton

headscale is an open source implementation of the server side of the
tailscale mesh network orchestration service.  The talk will cover
configuring the server and clients, how to create a simple mesh network
and setting up an exit node to push traffic through a private network.
