1. << Presentation (1) >>
**"Serving dynamic webpages in less then a millisecond"** by *John Fremlin*

John Fremlin will introduce the web application platform he
developed. The talk will cover issues like the greatest contributors to
latency and how he dealt with them, as well as other issues he addressed
in performance tuning.

[ [slides](http://tlug.jp/meetings/2008/11/serving-dynamic-webpages-in-less-then-a-millisecond_john-fremlin.pdf) ] [ [handout](http://tlug.jp/meetings/2008/11/serving-dynamic-webpages-in-less-then-a-millisecond_john-fremlin_handout.pdf) ]

**"designing and building high-performance HTTP systems"**

We will hold an open discussion about the design of high performance HTTP systems.
